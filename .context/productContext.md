# Product Context: Audit Insite Lab

Это сервис аудита сайтов, разработанный специально для маркетингового агентства.

## Основная цель
Автоматизировать процесс проверки сайтов клиентов по 11 ключевым параметрам и предоставлять структурированный отчет в формате PDF для последующей презентации клиенту.

## Ключевые возможности
- Анализ технических и SEO-параметров сайта.
- Проверка скорости загрузки и производительности.
- Визуальный аудит и оценка пользовательского опыта (UX).
- Генерация профессиональных PDF-отчетов.
